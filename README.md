<h2 align="center">REST_API для небольшой соц-сети</h2>


### Инструменты разработки

- Python >= 3.8
- Django Rest Framework
- Postgres

## Старт

#### 1) Создать образ

    docker-compose build

##### 2) Запустить контейнер

    docker-compose up
    
##### 3) Перейти по адресу
    http://0.0.0.0:5000/swagger/
